Performance Model Manager

PMM is a tool that has been developed to address issues surrounding the construction, maintenance and use of Functional Performance Models in a variety of parallel computing environments. It consists of three main features:

  • Firstly it implements the Geometric Bisection Building Procedure for multi-parameter FPMs, optimizing the construction of a problemâ€™s performance model.
  • Secondly, it permits a large number of problems to have their construction managed by implementing a flexible benchmarking scheduler, suitable for use where a queuing system does not exist.
  • Finally it provides access to the models in a variety of ways, allowing feedback from actual executions and providing tools to use the models in scheduling decisions.

This documentation describes the internal code that makes up PMM. The user guide can be found under docs/manual.pdf.

Further details on the application and optimised building can be found in:

Higgins, R., and A. Lastovetsky, "Managing the Construction and Use of Functional Performance Models in a Grid Environment", The 23rd IEEE International Parallel and Distributed Processing Symposium, Rome, Italy, May 25-29, 2009.

Lastovetsky, A., R. Reddy, and R. Higgins, "Building the Functional Performance Model of a Processor", Proceedings of the 21st Annual ACM Symposium on Applied Computing (SAC 2006), Dijon, France, ACM, April 23-27 2006.

pmmd is the main model building tool. It begins in pmm_main.c.

pmm_view, a viewer for models, is found in pmm_view.c.

GBBP the primary research algorithm implemented in this software can be found in pmm_selector.c. Functions of interest are:

  • benchmark() - main benchmark executing function, calling selection, execution and insertion
  • multi_gbbp_diagonal_select_new_bench() - main benchmark selection
  • multi_gbbp_bench_from_interval() - choosing a benchmark from an interval
  • multi_gbbp_insert_bench() - main benchmark insertion
  • process_interval() and the functions it calls - comparing model to new benchmark
